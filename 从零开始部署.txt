🎯 篮球训练助手 - 从零开始部署（超详细版）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⏱️ 预计用时：10-15分钟
😊 难度：超级简单（不需要任何编程知识）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  准备工作                                   ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

在开始之前，请确保：

□ 你有GitHub账号（没有的话去 github.com 注册）
□ 你有Netlify账号（没有的话去 netlify.com 用GitHub登录）
□ 你已经下载了最新的项目文件


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  第1步：下载并解压项目文件                  ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

1. 点击页面右上角的【下载】按钮
2. 会下载一个名为 basketball-trainer.zip 的文件
3. 找到这个文件，右键 → 解压到当前文件夹
4. 你会看到一个新的文件夹，里面有很多文件

⚠️ 重要：确保解压后的文件夹里包含一个名为 .gitignore 的文件
   （这个文件可能在某些电脑上看不见，这是正常的）


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  第2步：在GitHub创建新仓库                  ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

1. 打开浏览器，访问 https://github.com
2. 登录你的GitHub账号
3. 点击页面右上角的【+】号
4. 在下拉菜单中选择【New repository】（新建仓库）

5. 填写仓库信息：
   ┌────────────────────────────────────────┐
   │ Repository name（仓库名称）：            │
   │ basketball-trainer                      │
   │                                         │
   │ Description（描述）：                    │
   │ 篮球训练助手                             │
   │                                         │
   │ 选择：○ Public（公开）                   │
   │       ○ Private（私有）                 │
   │   → 选择 Public                         │
   │                                         │
   │ ⚠️ 不要勾选任何复选框！                   │
   │ □ Add a README file                    │
   │ □ Add .gitignore                       │
   │ □ Choose a license                     │
   │   → 这三个都不要勾选！                   │
   └────────────────────────────────────────┘

6. 点击底部绿色的【Create repository】按钮


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  第3步：上传项目文件到GitHub                ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

创建仓库后，你会看到一个新页面，上面有一些代码。
不用管那些代码，直接这样做：

1. 在页面中间找到一行蓝色的文字：
   "uploading an existing file"
   点击它

2. 你会进入上传文件的页面

3. 打开你刚才解压的文件夹（basketball-trainer）

4. ⚠️ 重要：选择文件夹里面的【所有文件和文件夹】
   不是拖整个 basketball-trainer 文件夹！
   
   应该看到这些文件：
   - App.tsx
   - main.tsx
   - index.html
   - package.json
   - vite.config.ts
   - tsconfig.json
   - netlify.toml
   - .gitignore （可能看不见，但确实存在）
   - components 文件夹
   - styles 文件夹
   - utils 文件夹
   - public 文件夹
   - 以及其他所有文件

5. 全选这些文件（按 Ctrl+A 或 Cmd+A）

6. 把选中的文件拖到GitHub的网页里
   （拖到写着"Drag files here"的地方）

7. 等待文件上传完成（可能需要1-2分钟）

8. 上传完成后，滚动到页面底部

9. 在"Commit changes"下面的文本框中输入：
   初始提交

10. 点击绿色的【Commit changes】按钮

11. 等待几秒钟，文件就上传好了！


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  第4步：检查文件是否上传正确                ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

上传完成后，你应该能在GitHub页面上看到：

✅ 应该看到的文件：
   - App.tsx
   - main.tsx
   - index.html
   - package.json
   - netlify.toml
   - components/
   - styles/
   - utils/
   - public/
   - 以及其他文件

❌ 不应该看到的文件夹：
   - node_modules （如果看到了，说明出错了）
   - dist （如果看到了，说明出错了）

如果看到了 node_modules 或 dist 文件夹：
→ 删除这个仓库，重新从第2步开始


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  第5步：在Netlify部署网站                   ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

1. 打开新的浏览器标签页，访问 https://netlify.com

2. 登录Netlify：
   - 点击【Log in】（登录）
   - 选择【Sign in with GitHub】（用GitHub登录）
   - 如果需要，授权Netlify访问你的GitHub

3. 登录后，点击【Add new site】按钮
   （或者【Sites】→【Add new site】）

4. 在下拉菜单中选择【Import an existing project】

5. 选择【Deploy with GitHub】

6. 如果是第一次，会弹出一个窗口要求授权：
   - 点击【Authorize Netlify】
   - 可能需要输入GitHub密码确认

7. 你会看到你的GitHub仓库列表
   找到【basketball-trainer】仓库，点击它

8. 进入配置页面，你会看到一些设置：
   
   ⚠️ 什么都不用改！
   
   应该自动显示：
   - Build command: npm install --legacy-peer-deps && npm run build
   - Publish directory: dist
   
   如果没有自动显示，手动填写上面的内容

9. 点击底部的【Deploy basketball-trainer】按钮
   （或者【Deploy site】按钮）

10. 现在开始部署了！


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  第6步：等待部署完成                        ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

1. 你会看到一个正在部署的页面

2. 页面顶部会显示：
   "Site deploy in progress"（网站部署中）

3. 等待3-5分钟（第一次部署会比较慢）

4. 部署过程中你会看到：
   - 正在下载代码...
   - 正在安装依赖...
   - 正在构建项目...
   - 正在发布网站...

5. 如果一切顺利，你会看到：
   ✅ "Published"（已发布）
   
   页面顶部会显示一个绿色的网址，类似：
   https://随机名字.netlify.app

6. 点击这个网址，就能看到你的应用了！

┃  第7步：测试你的应用                        ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

1. 点击Netlify给你的网址

2. 你应该能看到：
   - 橙色的标题"篮球训练助手"
   - 两个标签："指令管理"和"训练设置"
   - 漂亮的渐变背景

3. 如果看到了这些，恭喜你成功了！🎉

4. 试试添加一个训练指令：
   - 点击"指令管理"标签
   - 点击"添加指令"按钮
   - 输入"投篮"
   - 点击"添加"
   - 点击"录音"按钮测试录音功能


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  📱 添加到手机主屏幕                        ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

iPhone用户：
1. 用 Safari 浏览器打开你的网站
2. 点击底部中间的【分享】按钮
3. 滚动找到【添加到主屏幕】
4. 点击【添加】

Android用户：
1. 用 Chrome 浏览器打开你的网站
2. 点击右上角的【⋮】菜单
3. 选择【添加到主屏幕】或【安装应用】
4. 点击【添加】

添加后，你的手机桌面上会出现一个篮球训练助手的图标，
点击它就像打开普通APP一样！


┏━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  ❌ 如果遇到问题                            ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

问题1：部署失败（看到红色的Failed）
→ 打开【紧急修复指南.txt】查看解决方法

问题2：看到404错误
→ 可能是部署还没完成，等1-2分钟再试
→ 如果还是404，打开【紧急修复指南.txt】

问题3：网站是白屏
→ 按F12打开浏览器控制台查看错误
→ 尝试清除缓存重新部署

问题4：某个文件上传失败
→ 删除仓库重新开始（这是最快的方法）

   → 在Netlify的Site settings里可以改成自定义名字

2. 数据会丢失吗？
   → 不会！数据都保存在你的手机/电脑浏览器里

3. 需要付费吗？
   → 不需要！GitHub和Netlify都是免费的

4. 如何更新应用？
   → 在这里修改代码后重新下载
   → 在GitHub上传新文件覆盖旧文件
   → Netlify会自动重新部署

5. 如何分享给朋友？
   → 直接把网址发给朋友就可以了！
